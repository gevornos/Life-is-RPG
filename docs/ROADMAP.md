# Roadmap - План улучшений функционала

## Приоритет 1: Критичный функционал

### 1. Система монстров
- [ ] Появление монстров при пропуске ежедневных заданий
- [ ] Механика боя с монстрами (урон по атрибутам слабости)
- [ ] Награды за победу (XP, золото)
- [ ] Штрафы за поражение (потеря HP, XP)
- [ ] UI для отображения активных монстров
- [ ] Уведомления о появлении монстров

### 2. Система наград и прогрессии
- [ ] Начисление золота за выполнение заданий (сейчас только XP)
- [ ] Магазин предметов за золото
- [ ] Магазин премиум предметов за алмазы
- [ ] Система достижений
- [ ] Разблокировка новых аватаров при достижении уровней

### 3. Улучшение системы атрибутов
- [ ] Визуализация влияния атрибутов на игровой процесс
- [ ] Бонусы за высокие значения атрибутов
- [ ] Баланс штрафов за низкие атрибуты
- [ ] История изменения атрибутов (графики прогресса)

## Приоритет 2: Важный функционал

### 4. Улучшение системы привычек
- [ ] Возможность отметить отрицательное действие для негативных привычек
- [ ] Уведомления о привычках в определенное время дня
- [ ] Категории привычек (здоровье, работа, хобби и т.д.)
- [ ] Статистика по привычкам (графики серий, процент выполнения)

### 5. Улучшение системы ежедневных заданий
- [ ] Автоматический сброс completed_today в полночь
- [ ] Push-уведомления о невыполненных ежедневных заданиях
- [ ] Возможность переноса невыполненного задания на следующий день (за штраф)
- [ ] Недельные квесты с большими наградами

### 6. Улучшение системы задач (To-Do)
- [ ] Подзадачи (чекбоксы внутри задачи)
- [ ] Приоритеты задач (низкий, средний, высокий)
- [ ] Фильтры и сортировка (по дате, приоритету, атрибутам)
- [ ] Повторяющиеся задачи
- [ ] Архив выполненных задач

### 7. Социальные функции
- [ ] Гильдии (группы пользователей)
- [ ] Общие цели гильдии
- [ ] Рейтинг игроков по уровню
- [ ] Возможность помочь другу (отправить HP, золото)
- [ ] Совместные монстры (рейд-боссы)

## Приоритет 3: Желательный функционал

### 8. Аналитика и статистика
- [ ] Дашборд с общей статистикой прогресса
- [ ] Календарь активности (как на GitHub)
- [ ] Графики изменения атрибутов по времени
- [ ] Еженедельные/ежемесячные отчеты
- [ ] Экспорт статистики в PDF/CSV

### 9. Кастомизация
- [ ] Темы оформления (темная/светлая уже есть, добавить другие)
- [ ] Кастомные цвета атрибутов
- [ ] Создание собственных аватаров (загрузка изображений)
- [ ] Кастомные звуки уведомлений

### 10. Игровые механики
- [ ] Система крафта (создание предметов из ресурсов)
- [ ] Питомцы (компаньоны, дающие бонусы)
- [ ] Случайные события (рандомные квесты, бонусы)
- [ ] Босс месяца (особо сильный монстр)
- [ ] Сезонные события и ограниченные награды

### 11. Интеграции
- [ ] Интеграция с календарем телефона
- [ ] Интеграция с фитнес-трекерами (Google Fit, Apple Health)
- [ ] Интеграция с Notion/Todoist для импорта задач
- [ ] Web-версия приложения
- [ ] Desktop-приложение (Electron)

## Ближайшие шаги (Sprint 1)

1. **Система монстров** - основная игровая механика, без неё мало мотивации
2. **Начисление золота** - сейчас оно не начисляется, только отображается
3. **Автоматический сброс ежедневных** - критично для корректной работы
4. **Магазин предметов** - нужно куда-то тратить золото
